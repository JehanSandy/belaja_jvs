menghubugkan vsCode dengan repository baru di github
1. git init
2. git remote (alamat url)
3. git add .
4. git commit m "upload pertama"
5. git push origin master

git config --global user.name "My Name"
git config --global user.email "myemail@example.com"
git config --list (cek username kita)
